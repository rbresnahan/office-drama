# Simple Choose Your Own Adventure Sandbox
A tiny plain-JavaScript template you can reuse for different stories.

---

## Goal

Build a very small browser-based adventure that supports:

- a start screen
- story text
- choices
- consequences
- branching paths
- multiple endings
- restart

No frameworks. No build tools. No nonsense.

---
